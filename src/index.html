.Container.

Container is the outer box (or) the main box i,e..parent element of all other elements

A container at larger screen size has a width of 1140 px and reduces w.r.t screen size. A container is fully responsive.

By default a container has a left and right padding of 15px

Responsive Grid

Bootstrap provides 5 different @media queries w.r.t screen size.They define screen sizes in folloowing ways.

  1. Extra large (xl) → screens whose screen size is >= 1200px

    2. xl screen container width is 1140px

  2. Large (lg) → screens whose screens size is >=992px

    3. lg screen container width is 960px

  3. Medium (md) → screens whose screens size is >= 768px

    4. md screen container width is 720px

  4. Small (sm) → screens whose screen size is >=576px

    5. sm screen container width is 540px

  5. Extra small (xsm) → screens whose screen size is <=575px

    6. xsm screen container width is auto
